# HeyMate Product Definition

## 1. アプリ概要
- アプリ名: HeyMate
- コンセプト: 合トレマッチングアプリ
- 目的: 一人では続かない筋トレを、仲間との合トレで習慣化する
- 非目的: 恋愛・出会いを主目的としたマッチング

## 2. プロダクトビジョン
HeyMateは、トレーニング継続の難しさと孤独感を解消し、
「気軽に誘い合えるジム仲間」と「継続できる運動習慣」を作る。

## 3. ターゲット層
- メイン: 20代男性の社会人トレーニー（初級〜中級）
- サブ: 継続モチベーションに波がある一人トレーニー
- 利用シーン: 平日夜・休日のジム通い、合トレ募集/参加

## 4. ペルソナ

### ペルソナ1: 山東 拓（23）
- 社会人2年目 / 一人暮らし / 上京後コミュニティ不足
- 課題: 一人トレのモチベ低下、声かけのハードル、孤独感
- 求める価値: 合トレ相手、休日の充実、継続しやすさ

### ペルソナ2: 川門さん（29）
- 社会人4年目 / 中級者 / 忙しく習慣が切れやすい
- 課題: 誰かとの約束がないと運動が後回しになる
- 求める価値: 同レベルマッチ、予定共有、軽いコミュニティ

### ペルソナ3: 笹井さん（26）
- 初級〜ライト中級 / 成長意欲が高い
- 課題: フォーム不安、メニューのマンネリ、成果実感不足
- 求める価値: 経験者との接点、学習機会、効率的な成長

## 5. 提供価値
1. 合トレ相手の発見
2. 継続モチベーションの維持（約束による行動促進）
3. ジムコミュニティ形成
4. 孤独感の軽減
5. トレーニング体験の質向上

## 6. 成功指標（初期案）
- 週次継続率（W2/W4）
- マッチ成立率
- マッチ後チャット開始率
- 週1回以上の合トレ実施率

## 7. 価格/収益モデル（初期案）
- 月額: 980円
- 課金対象案: 機能制限解除、優先表示、詳細フィルタなど

## 8. ブランド/デザイン方針
- メインカラー: `#6EC6FF`
- アクセント: `#1A237E`（代替 `#2C3E50`）
- ベース: `#FFFFFF`
- サブ: `#F5F7FA` / `#E1E8ED`
- トーン: 爽やか・クリーン・信頼感

## 9. 制約と前提
- 安全性・法令対応をMVP段階から実装
- マッチング機能は不正利用対策を優先
- 個人情報は最小取得を原則とする
